JavaScript (validação) + PHP (processamento)

ℹ️ Como funciona:

1. JavaScript (navegador): Valida se os campos estão preenchidos, se email é válido, idade é número.

2. PHP (servidor): Processa os dados, valida novamente (segurança), salva no banco de dados ou arquivo.

3. Os dados são enviados com method="POST" para o arquivo processar.php
